# Welcome to DELI-cious, The Last Deli Point of Sale you will ever need!
### It is simple to use and understand, here are all the details
- Create a custom sandwich easily and quickly
- Add a drink, chips or another sandwich
- Once everything is selected you can checkout and a receipt will automatically be generated.
